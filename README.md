# lita-odot-tripcheck



## Installation

Add lita-odot-tripcheck to your Lita instance's Gemfile:

``` ruby
gem "lita-odot-tripcheck"
```


## Configuration

Lita.configure do |config|
  config.handlers.odot_tripcheck.uri = ''  # Ensure you put 
  config.handlers.odot_tripcheck.uid = 'xxxxxxxxx'
end
